# AI SDLC Maturity Tools

This repo contains four standalone HTML tools for assessing and tracking AI adoption in the Software Development Lifecycle (SDLC):

- **AI_SDLC_Interactive.html**: Interactive self-assessment. Open in a browser, answer questions, and view your AI adoption roadmap. Data is saved in your browser (localStorage).
- **AI_SDLC_Maturity_Questionnaire.html**: Static version of the assessment questions. Open in a browser to review all questions and options.
- **AI_SDLC_Quest_Board.html**: Quest board for tracking progress and tasks. Open in a browser to view and manage quests.
- **AI_SDLC_Maturity_Matrix.html**: Reference matrix for maturity levels and criteria. Open in a browser to see requirements for each level.

## Recommended Usage
1. Start with `AI_SDLC_Maturity_Questionnaire.html` for a simple, static overview of all survey questions and options.
2. Once familiar, use `AI_SDLC_Interactive.html` for an interactive experience with a visual roadmap that shows your progress and next steps.
3. Use `AI_SDLC_Quest_Board.html` to track tasks or quests related to your AI SDLC maturity journey.
4. Refer to `AI_SDLC_Maturity_Matrix.html` for a detailed reference of maturity levels and their requirements.

## How to Use
- Open any HTML file in your web browser. No server or build step required.
- Data is stored locally in your browser (no backend, no sync).
- No installation or dependencies.

## Notes
- For self-assessment only. No data is sent or shared.
- Each tool is independent; there is no cross-file data sharing.
- Reset or clear browser storage to start over.
